# N-Queens Puzzle
A web-based N-Queens puzzle built with HTML, CSS, and JavaScript.

## How to run
Just open index.html in any browser.

## Features
- Choose board size from 4 to 10
- Place queens manually by clicking cells
- Conflict detection highlights attacking queens in red
- Show Solution button uses DFS with backtracking to solve the board

## Algorithm
Depth First Search (DFS) with Backtracking.
Places one queen per row, checks if the position is safe using column and diagonal checks, backtracks if stuck.

## Technologies
HTML, CSS, JavaScript
